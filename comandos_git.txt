git init . 		-> Cria um repo git na pasta atual
git status 		-> Mostra o status do trabalho
git add nome_arquivo 	-> Adiciona arquivo parta ser commitado
git commit -m "msg" 	-> Faz commit do que estava para ser salvo
git log 		-> Mostra todo o histórico de commit

## FLUXO DE TRABALHO
1. Criar o diretório (pasta) do projeto
2. iniciar o git dentro fo diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. volta para o passo 3
